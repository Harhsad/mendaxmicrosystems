Overview

The MDX-T2U server delivers high performance and scalability in a rack mount 2U form factor. The system provides a wide range of features and functionalities and vast storage capabilities and performance.

The MDX-T2U server is designed for performance and virtualized environments. Expandable computer network hardware system, easy to manage and ready to deploy.
